# 知行 KnowDo - 页面原型

> 企业级AI知识库平台 · 页面交互原型 v1.0

基于《企业级AI知识平台.md》PRD文档设计的HTML页面原型，参考 MaxKB 设计风格，融合金融企业级专业设计语言。

## 📋 页面清单

| 页面 | 文件 | 对应PRD功能 |
|:---|:---|:---|
| 🏠 **平台首页** | `index.html` | 统计概览、快捷入口、热门知识Top10、最新知识时间线、通知中心 |
| 📚 **知识分类浏览** | `knowledge-browse.html` | 三级分类树（F-11）、多维筛选排序（F-14）、知识卡片列表、悬停预览（F-15） |
| 📄 **知识详情** | `knowledge-detail.html` | AI摘要（F-30）、正文展示、附件下载、评论点赞（F-19）、收藏分享（F-18/F-20）、推荐（F-21） |
| ✏️ **知识创建/编辑** | `knowledge-create.html` | 6种类型选择（F-01）、模板表单（F-02）、富文本编辑器（F-03）、AI辅助创作（F-31）、草稿保存（F-05）、提交审核（F-04） |
| 🤖 **模型配置** | `model-config.html` | 三类模型管理（M-01/M-02）、接入配置、连接测试、启用禁用、向量化策略（M-06） |

## 🚀 使用方式

### 方式一：直接打开（推荐）
```bash
# 在浏览器中直接打开任意HTML文件
open prototypes/index.html
```

### 方式二：本地服务器
```bash
cd prototypes
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```

## 📁 目录结构

```
prototypes/
```

## 🎨 设计规范

### 色彩系统
| 角色 | 色值 | 用途 |
|:---|:---|:---|
| 主色 | `#1a56db` | 按钮、链接、选中态 |
| 主色深 | `#1e40af` | hover态 |
| 背景白 | `#ffffff` | 卡片、内容区 |
| 背景灰 | `#f0f2f5` | 页面底色 |
| 成功 | `#10b981` | 通过、在线 |
| 警告 | `#f59e0b` | 待审核、提醒 |
| 错误 | `#ef4444` | 驳回、必填提示 |

### 布局
- 顶部导航：56px 深蓝底色（`#0f172a`），固定顶部
- 左侧导航：220px 白色背景，可折叠
- 内容区：自适应宽度，最大1400px

### 字体
- 标题：24px / 700
- 副标题：20px / 600
- 正文：14px / 400
- 辅助：12px / 400

## 🔧 技术实现

| 项目 | 说明 |
|:---|:---|
| **框架** | 纯HTML5 + CSS3 + 原生JavaScript（ES6） |
| **样式** | Tailwind CSS CDN + 自定义CSS变量 |
| **图标** | Emoji + 内联SVG（零外部依赖） |
| **数据** | 静态JSON模拟数据（可替换为真实API） |
| **兼容** | Chrome 100+ / Edge 100+ / Firefox 100+ / Safari 16+ |

## 📝 交互说明

### 已实现的交互
- ✅ 左侧导航高亮当前页 + 页面间跳转
- ✅ 分类树展开/折叠
- ✅ 知识卡片悬停预览浮窗
- ✅ 点赞/收藏实时切换
- ✅ 筛选条件联动（类型/标签/时间/排序）
- ✅ 步骤条进度可视化（创建知识）
- ✅ 富文本编辑器工具栏
- ✅ AI辅助工具栏（选中文本弹出）
- ✅ 评论发表与回复
- ✅ 分享弹窗（复制链接/二维码/站内信）
- ✅ Toast消息提示
- ✅ Modal弹窗（确认删除/提交成功）
- ✅ 模型连接测试动画
- ✅ 通知中心下拉
- ✅ 向量化策略滑块联动
- ✅ 草稿自动保存提示

### 模拟交互（演示用）
- 🔄 AI摘要生成、AI辅助创作（模拟延迟 + Toast反馈）
- 🔄 文件上传（Toast提示模拟）
- 🔄 模型测试连接（随机成功/失败）
